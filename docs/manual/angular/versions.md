# Angular v18-v20 Changes (Summary)

Short, high-signal highlights with links to official release posts.

## v20 (May 28, 2025)
- Stabilized `effect`, `linkedSignal`, and `toSignal`, plus incremental hydration and route-level render mode config.
- Zoneless change detection is in developer preview.
- Angular DevTools debugging improved and integrated into Chrome DevTools.
- Language service adds host bindings type checking; template HMR is on by default.

## v19 (Nov 19, 2024)
- Incremental hydration in developer preview.
- Route-level render modes and prerender params for SSR.
- New/expanded reactivity APIs: `linkedSignal` and `resource`.
- HMR for styles by default; template HMR behind a flag.
- Standalone defaults + strict standalone enforcement.

## v18 (May 22, 2024)
- Experimental zoneless change detection.
- Material 3, deferrable views, and built-in control flow are stable.
- SSR improvements (i18n hydration support, better debugging, event replay).
- Angular.dev becomes the official docs home.
